# Nemovitosti

Tato složka obsahuje všechny nemovitosti rozdělené do tří kategorií:
- **Prodej/** - Aktivní nabídky prodeje
- **Pronajem/** - Aktivní nabídky pronájmu  
- **Hotovo/** - Prodané nebo pronajaté nemovitosti

## Struktura složky nemovitosti

Každá nemovitost musí obsahovat:
1. `data.md` - Markdown soubor s informacemi o nemovitosti
2. `img00.jpg` - Hlavní náhledový obrázek
3. `img01.jpg, img02.jpg, ...` - Další obrázky do galerie

Příklad: `nemovitosti/Prodej/Dum_U_Prehrady_Cheb/`
